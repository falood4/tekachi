# Tekachi

Tekachi is a mobile-first placement preparation platform built using Flutter, designed to help students systematically prepare for aptitude tests, technical interviews, HR interviews, and leverage an AI mentor for guided learning and mock preparation.

## Overview

Tekachi consolidates all major placement preparation components into a single application:

Aptitude training and testing

Technical interview preparation

HR interview question bank

AI-powered mentor assistance

Clean, structured, multi-step placement workflow

The application is optimized for students preparing for campus placements and entry-level technical roles.
